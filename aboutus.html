Purpose of Lawpedia

We need some information for various day to day activities. What if all the information we need for a particular task is available at one place? It will be a blessing in disguise as it will be both time saving and we can get the complete information. This project provides a platform where all the lawyers of city are listed with detailed description. This description includes from reviews to personal info and it also provides a easy way to contact them. By using it, an inexperienced person in law field can also choose a most appropriate option for his/her problem.
